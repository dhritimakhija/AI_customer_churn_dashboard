# AI_customer_churn_Dashboard
This dashboard uses a Random Forest machine learning model (≈79% accuracy) to identify customers likely to churn and segment them into High, Medium, and Low risk categories. It also calculates Revenue at Risk to highlight the potential financial impact of customer churn.

The dashboard enables interactive analysis across contract types, services, payment methods, and customer demographics to uncover patterns driving customer attrition.

Key insights from the analysis show that month-to-month contracts have the highest churn rate, customers using electronic check payments demonstrate higher churn risk, and customers without tech support or online security services are more likely to leave.

This project demonstrates how predictive analytics combined with interactive BI dashboards can help businesses move beyond historical reporting to anticipate customer behavior and improve retention strategies.

Technologies used: Python, Pandas, Scikit-learn, Feature Engineering, Power BI, DAX, Data Modeling
